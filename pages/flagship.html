16-Week Curriculum Breakdown

1

Phase 1: Data Engineering & Foundations

Weeks 1-4 | Focus: Building robust data pipelines

Week 1: Advanced Python & Vectorization

  • • Writing efficient NumPy/Pandas code
  • • Type hinting and unit testing for data science

Week 2: SQL & NoSQL for AI

  • • Complex joins and window functions
  • • Vector databases (Pinecone/Weaviate) for RAG

Week 3: Data Cleaning at Scale

  • • Handling missing data, outliers, class imbalance
  • • Building reproducible pipelines with Airflow

Week 4: Project 1 - The Data Pipeline

Build an automated ETL pipeline that scrapes web data, cleans it, and stores it in a vector database.

2

Phase 2: Machine Learning & Deep Learning

Weeks 5-10 | Focus: From Scikit-Learn to Transformers

Week 5: Supervised Learning Deep Dive

  • • Gradient Boosting (XGBoost/LightGBM)
  • • Hyperparameter tuning with Optuna

Week 6: Neural Networks with PyTorch

  • • Tensors, autograd, and backpropagation
  • • Building MLP and CNN architectures from scratch

Week 7: Computer Vision

  • • Transfer learning (ResNet, EfficientNet)
  • • Image segmentation and object detection (YOLO)

Week 8: NLP & LLMs

  • • Transformers architecture
  • • Fine-tuning Llama/Mistral with HuggingFace PEFT and LoRA

Week 9: Generative AI & RAG

  • • Building RAG systems (LangChain)
  • • Prompt Engineering vs. Fine-tuning trade-offs

Week 10: Project 2 - The Custom LLM

Fine-tune a small LLM on a specific legal or medical dataset and expose it via a chat interface.

3

Phase 3: MLOps & Deployment

Weeks 11-14 | Focus: The "Engineering" in AI Engineering

Week 11: Containerization

  • • Dockerizing ML models
  • • Multi-stage builds for GPU-enabled containers

Week 12: Model Serving & API Design

  • • Building high-performance APIs with FastAPI
  • • Batch vs. Real-time inference patterns

Week 13: Cloud Deployment (AWS/GCP)

  • • Deploying to AWS Lambda vs. SageMaker
  • • Setting up CI/CD pipelines (GitHub Actions)

Week 14: Monitoring & Observability

  • • Tracking model drift
  • • Tools: MLflow, Prometheus, Grafana
